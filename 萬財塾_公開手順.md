# 萬財塾サイト 公開手順書

## 全体の流れ（所要時間：約15分）

1. GitHubでリポジトリを作成
2. Personal Access Token を発行
3. HTMLファイルをアップロード
4. GitHub Pages を有効化
5. サイトからGitHub接続設定を行う

---

## ステップ1　GitHubでリポジトリを作成

1. [https://github.com/new](https://github.com/new) にアクセス
2. 以下のように入力する
   - Repository name: `manzai-juku`
   - Visibility: **Public**（公開）を選択
3. 「Create repository」ボタンをクリック

---

## ステップ2　Personal Access Token を発行

1. [https://github.com/settings/tokens/new](https://github.com/settings/tokens/new) にアクセス
2. 以下を設定する
   - Note: `萬財塾`
   - Expiration: `1 year`（1年間有効）
   - Select scopes: **public_repo** にチェックを入れる
3. 「Generate token」ボタンをクリック
4. 表示されたトークン（`ghp_xxxx...`）をコピーして安全な場所に保存する
   ⚠️ このページを閉じると二度と確認できません

---

## ステップ3　HTMLファイルをアップロード

1. 作成したリポジトリのページを開く
2. 「uploading an existing file」のリンクをクリック
3. `萬財塾.html` をドラッグ＆ドロップ
4. ファイル名を `index.html` に変更する（URLがきれいになります）
5. 「Commit changes」ボタンをクリック

---

## ステップ4　GitHub Pages を有効化

1. リポジトリの「Settings」タブをクリック
2. 左メニューの「Pages」をクリック
3. Source: 「Deploy from a branch」を選択
4. Branch: `main` / `/(root)` に設定
5. 「Save」をクリック
6. しばらく待つと以下のURLでサイトが公開される

```
https://（あなたのGitHubユーザー名）.github.io/manzai-juku/
```

---

## ステップ5　サイトからGitHub接続設定を行う

1. 公開されたURLをブラウザで開く
2. 右上の「⚙ GitHub設定」ボタンをクリック
3. 以下を入力する
   - GitHubユーザー名: あなたのGitHubのユーザー名
   - リポジトリ名: `manzai-juku`
   - Personal Access Token: ステップ2でコピーしたトークン
4. 「接続して保存」をクリック
5. ヘッダーに「● GitHub 接続中」と表示されれば完了

---

## 日常の使い方

### セッションを追加するとき
1. サイトにアクセス
2. 右上「＋ 新規追加」ボタンをクリック
3. 回数・日付・テーマ・カテゴリを入力して「追加する」
4. GitHubに自動保存される（どの端末からでも反映される）

### 次回テーマをAIに提案してもらうとき
1. 右上「🤖 次回テーマをAIに提案」ボタンをクリック
2. 「📋 クリップボードにコピー」をクリック
3. Claude（https://claude.ai）のチャットに貼り付けてEnter

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| 「GitHub API エラー: 404」 | リポジトリ名の入力ミス | 設定を確認して再入力 |
| 「トークンが無効です」 | トークンの入力ミスまたは期限切れ | GitHub で新しいトークンを発行 |
| サイトが開かない | Pages 有効化が完了していない | 5〜10分待ってから再アクセス |
| 追加したセッションが消えた | ブラウザを変えたため | GitHub 設定を再入力すれば同期される |

---

## セキュリティについて

- Personal Access Token はこのブラウザにのみ保存され、GitHub との通信にのみ使用されます
- サイト自体は Public（誰でも閲覧可能）ですが、セッションの追加はトークンを持つ人だけ可能です
- トークンが漏れた場合は GitHub で即座に無効化できます（Settings → Tokens → Revoke）
