⚙ GitHub 接続設定
1

GitHubでリポジトリを作成

github.com/new にアクセスして新しいリポジトリを作成します。
名前の例: manzai-juku　／　Public（公開）で作成してください。
2

Personal Access Token を発行

github.com/settings/tokens/new を開き、
・Token name: 萬財塾
・Expiration: 1 year
・スコープ: public_repo にチェック
→ 「Generate token」ボタンを押してトークンをコピー
3

GitHub Pages を有効化（ホスティング）

リポジトリの Settings → Pages → Source を「Deploy from a branch」→ branch: main に設定。
URLが発行されたら、このHTMLファイルをそのリポジトリにアップロードしてください。
